Step 4.2¶

Split the data into training and testing datasets

Using the sklearn.model_selection.train_test_split() function to split the training data, feature set and target set (i.e. X and Y), into two subsets for training and testing, i.e. X_train, X_test, Y_train, Y_test.

Set test_size to 0.3. test_size is the proportion of the dataset to include in the test split. Set random_state to 0. This integer value is the seed used by the random number generator. Set stratify to stratify the dataset based on the target labels (i.e. Y).
